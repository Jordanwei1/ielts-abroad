## 🎯 如何使用

### 1. 雅思备考路径

```
Step 1: 评估当前水平
  ↓
Step 2: 制定学习计划
  ↓
Step 3: 按模块学习
  - 听力: 场景词汇 → 题型技巧 → 真题训练
  - 阅读: 替换词库 → 长难句 → 限时训练
  - 写作: 背诵模板 → 积累素材 → 限时写作
  - 口语: Part1/2/3 → 录音练习 → 模拟考试
  ↓
Step 4: 模考+复盘
  ↓
Step 5: 考试
```

### 2. 留学申请路径

```
提前1年: 规划+选校
  ↓
提前9个月: 语言考试
  ↓
提前6个月: 文书写作
  ↓
提前3个月: 提交申请
  ↓
等待offer
  ↓
签证+行前准备
```

### 3. 海外生活路径

```
出国前: 学会10道拿手菜
  ↓
到达后: 熟悉超市+中超
  ↓
日常: 自己做饭省钱
  ↓
进阶: 学习西餐+烘焙
```

## 💡 学习建议

### 雅思备考

1. **每日训练**: 坚持每天学习2-3小时
2. **SOP训练**: 按照固定流程训练
3. **真题为主**: 剑桥真题反复做
4. **模考复盘**: 每周模考+错题分析

### 留学申请

1. **提前规划**: 至少提前1年准备
2. **多方咨询**: 学长学姐+中介
3. **文书打磨**: 多次修改,突出亮点
4. **保底冲刺**: 申请要有梯度

### 海外生活

1. **学会做饭**: 省钱又健康
2. **社交融入**: 多参加活动
3. **理财规划**: 控制开支
4. **持续学习**: 投资自己
